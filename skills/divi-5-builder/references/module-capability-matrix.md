# Divi 5 Module Capability Matrix

Use this reference when choosing a Divi-native module or system before proposing custom code.

## Table Of Contents

- Content And Editorial
- Layout And Components
- Dynamic Content
- Navigation
- Commerce
- Forms And Conversion
- Interactivity

## Content And Editorial

| Need | Prefer | Notes |
| --- | --- | --- |
| Headings and simple text | Heading, Text | Use presets and variables for typography. |
| Images and media | Image, Video, Gallery, Slider | Use image presets, aspect ratio, and framing. |
| Lists with icons | Icon List, Blurb, Text | Prefer Icon List or Blurb when icon styling matters. |
| Long-form posts | Theme Builder plus Post Content | Keep post body editable in WordPress. |
| Blog feed quickly | Blog module | Use when fixed Blog module layouts are enough. |
| Custom blog/resource cards | Loop Builder | Build the card with normal modules and dynamic content. |
| Dated sequence | Timeline | Use Timeline Items and Even option groups for alternating layouts. |

## Layout And Components

| Need | Prefer | Notes |
| --- | --- | --- |
| One-axis alignment | Flexbox | Good for nav rows, buttons, card internals, vertical stacks. |
| Two-axis placement | CSS Grid | Good for feature grids, galleries, editorial layouts, dashboards. |
| Reusable card/group | Group module | Apply presets and Admin Labels. |
| Carousel of grouped cards | Group Carousel | Verify responsive dots/slides after update. |
| Reusable styles | Variables and Presets | Build system before repeating styles. |

## Dynamic Content

| Need | Prefer | Notes |
| --- | --- | --- |
| Post/product/archive templates | Theme Builder | Assign templates by content type or condition. |
| Repeating post/product/custom data | Loop Builder | Use dynamic fields and query controls. |
| ACF/repeater-driven layouts | Loop Builder | Verify field availability and frontend output. |
| Shared hidden/reusable content | Canvases and Canvas Portal | Good for popups, sidebars, and inserted panels. |

## Navigation

| Need | Prefer | Notes |
| --- | --- | --- |
| WordPress menu | Menu module | Keep menu items in WordPress menus. |
| Custom simple nav items | Link module | Good for semantic, preset-driven navigation. |
| Dropdown navigation | Dropdown module | Combine with Link modules and interactions as needed. |
| Dynamic menus | Menu Loops | Use when menu structures need dynamic repetition. |
| Mobile/off-canvas nav | Canvas plus Interactions | Provide clear touch and close behavior. |

## Commerce

| Need | Prefer | Notes |
| --- | --- | --- |
| Product grid | Woo Products or Loop Builder | Use Loop Builder for custom card structure. |
| Product page | Theme Builder plus Woo modules | Test variable products and stock states. |
| Checkout | Woo Checkout modules in Theme Builder | Use Billing, Shipping, Information, Details, Payment, and Notice modules. |
| Cart/account flow | Woo modules and WooCommerce settings | Test logged-in and guest flows. |

## Forms And Conversion

| Need | Prefer | Notes |
| --- | --- | --- |
| Native contact form | Contact Form | Use Input, Checkbox, Radio presets. |
| Email signup | Email Optin | Verify provider connection and field states. |
| Contact Form 7 styling | Contact Form 7 module | Use native Divi styling instead of shortcode CSS where available. |
| Form field consistency | Field Option Group Presets | Include focus and checked states. |

## Interactivity

| Need | Prefer | Notes |
| --- | --- | --- |
| Toggle, reveal, popup | Interactions | Define trigger, effect, and target clearly. |
| Off-canvas content | Canvases | Good for menus, slide-ins, popups, staged content. |
| Hover-only effect | Interactions with touch alternative | Always provide a non-hover path for mobile. |
| Scroll animation | Scroll Effects or Interactions | Keep motion restrained and accessible. |
